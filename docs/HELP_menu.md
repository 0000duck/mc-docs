## Forums

Opens the [MatterControl
section](http://forums.matterhackers.com/category/20/mattercontrol) of
the MatterHacker forums in a web browser.

## Wiki

Opens [the main page of this wiki](Main_Page "wikilink") in a web
browser.

## Guides and Articles

Opens the [MatterControl section of MatterHackers
articles](http://www.matterhackers.com/topic/mattercontrol).

## Release Notes

Opens the [Release Notes](Release_Notes "wikilink") page of this wiki in
a web browser.

## Report a Bug

Opens the message window which will send submitted messages to
support@matterhackers.com.

![Report\_a\_Bug.png](http://wiki.mattercontrol.com/images/2/20/Report_a_Bug.png "http://wiki.mattercontrol.com/images/2/20/Report_a_Bug.png")

## Check For Update

Opens the **Check for Update** window.

![Check\_for\_Update.png](http://wiki.mattercontrol.com/images/e/eb/Check_for_Update.png "http://wiki.mattercontrol.com/images/e/eb/Check_for_Update.png")
